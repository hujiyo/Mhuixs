### 项目名： Mhuixs 

别名：基于内存的非关系型数据库或关系型数据库 或 基于内存的高维度数据关系存储或数据存储库

介绍：学习国防安全后我深受未来战争信息化趋势触动。信息化战场依托战场实时数据实现对战场精准感知与操控。我觉得未来中国国防将依赖于高性能、分布式且具安全性的数据库管理系统甚至是操作系统

### 未来特点：

1、基于内存的高速数据库，在内存贬值的未来Mhuixs即将成为主流

2、支持基于主外键的关系型或非关系型的表间管理，支持表组管理，支持宏观的权限管理。

3、键值对基于muemurhash算法进行分列扩散，效率及速度远超其他查询结构

4、支持键关系的基础创建和键间联系强度的设定，基于此可以打造超维度神经网络式的键间关系

5、支持键对表、表组、键值对库、栈结构的引用，实现你想要让谁和谁有关系，它们之间就可以产生关系

6、支持软多线程命令的事务处理

7、支持分布式内存链接

8、..........
