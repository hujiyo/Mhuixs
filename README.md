#### 介绍
学习国防安全后我深受未来战争信息化趋势触动。信息化战场依托战场实时数据实现对战场精准感知与操控。我觉得未来中国国防将依赖于高性能、分布式且具安全性的数据库管理系统甚至是操作系统【所以趁空闲时间想写个基于内存的数据库。现在这项目就我一人，没有压力（没老师），我已经实现表功能了，想找几个人和我一起合作】
